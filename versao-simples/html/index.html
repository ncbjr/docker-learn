🎯 Objetivo desta versão:

Aprender os conceitos básicos do Docker de forma simples e direta.

📦 O que você aprendeu até aqui:

  • ✅ Como rodar um container: docker run
  • ✅ Mapear portas: -p 8080:80
  • ✅ Montar volumes: -v ./html:/usr/share/nginx/html
  • ✅ Rodar em background: -d

🚀 Próximos passos:

  1. Edite este arquivo HTML e veja a mudança instantânea
  2. Acesse a API em: http://localhost:8081
  3. Experimente parar e iniciar containers
  4. Explore os logs dos containers

📚 Comandos essenciais:

docker ps - Ver containers rodando

docker logs [nome] - Ver logs

docker stop [nome] - Parar container

docker-compose up -d - Subir tudo

docker-compose down - Derrubar tudo
