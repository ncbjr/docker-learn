🚀 API Backend

✅ Servidor API rodando na porta 8081

📍 Endpoints Disponíveis:

GET / - Esta página

GET /status - Status da API

GET /usuarios - Lista de usuários

🔗 Comunicação entre Containers:

Este container pode ser acessado pelos outros através do nome: api

Exemplo: curl http://api/status

🌐 Acesso Externo:

Mapeado para: localhost:8081

Rede: aula-network

Container: aula-api | Imagem: httpd:alpine
